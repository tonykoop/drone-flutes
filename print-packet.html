3 · Design Intent

  • Open-open pipe + NAF K2 empirical correction
  • A=440 Hz concert pitch
  • 68°F reference temperature
  • Octagonal body, 8 facets, walnut with hard-maple CNC inlay
  • Drone offset = global var; flips all 4 flutes in one edit
